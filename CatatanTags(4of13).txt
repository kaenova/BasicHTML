Tag of <HTML>
tag ada yang berpasangan dan ada yang tidak berpasangaan

Struktur dasar HTML:
<!doctype html>
<html>
    <head>
        <title></title>
    </head>

    <body>
    
    </body>
</html>

Tag <head> bisa ditambahkan
    - Judul Halaman
    <title> </title>
    - CSS
    <style> </style>
    - javascript
    <script> </script>
    - metadata
    <meta> </meta>

Tag <body> bisa ditambahkan
    - teks
    <h1>, <h2>, <h3>, ..., <h6>, <p>, ...
    - pendukung teks
    <br>, <hr>, <em>, <strong>, ...
    - gambar
    <img>
    - hyperlink
    <a> (a = anchor)
    - list (bullets & numbering)
    <ul> (unordered list), <ol> (ordered list), <li> (list item), <dl> (definition list), <dt>, <dd>
    - table
    <table>, <thead>, <tbody>, ...
    - form
    <form>, <input>, <select>, <button>
    - script (java script)
    <script>
    - object
    <object>
    - frouping
    <div>, <span>

Komentar
    <!-- [Isi Komentar] -->
